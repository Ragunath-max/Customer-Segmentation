# Customer Segmentation using K-Means Clustering

## Project Overview
This project uses the K-Means Clustering algorithm to segment customers based on their Annual Income and Spending Score.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Scikit-learn

## Dataset
Mall Customers Dataset

## Algorithm
K-Means Clustering

## Features
- Data Preprocessing
- Elbow Method
- Customer Clustering
- Cluster Visualization

## Output
Customers are grouped into 5 different segments based on purchasing behavior.
